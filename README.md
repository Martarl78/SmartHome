🏡 Mi Hogar Inteligente
Proyecto sencillo de un panel para visualizar información de un hogar inteligente. Incluye widgets de temperatura, luces, humedad, cámaras, seguridad, música y recordatorios. El diseño es responsive y permite cambiar entre modo claro y modo oscuro.

🚀 Cómo usarlo
Descarga el proyecto y abre el archivo index.html en cualquier navegador. No necesita instalación ni dependencias.

🧩 Tecnologías
HTML y CSS.

📂 Contenido
El proyecto incluye:
• Carpeta  con las imágenes del dashboard
• Index.html
• EstiloIndex.css

🌗 Modo oscuro
El panel incluye un interruptor para cambiar entre modo día y modo noche usando variables CSS.
